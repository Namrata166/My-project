# 🐍 Snake Game (Python)

## 🎮 About the Project

This is a classic **Snake Game** built using **Python**.
The game is inspired by the old Nokia mobile snake game that many of us enjoyed.

I created this project to practice **Python programming, game logic, and UI design**.
---
## ✨ Features

* 🎯 Classic snake gameplay
* ⬆️ Smooth controls (keyboard-based)
* 🍎 Food generation system
* 📈 Score tracking
* 💀 Game over detection
* 🎨 Simple and clean UI
---
## 🛠️ Tech Stack
* Python
* Turtle 
---
## 🚀 How to Run the Game
1. Clone the repository:
```
git clone https://github.com/Namrata166/My-project.git
```
2. Navigate to the folder:```
cd My-project
```
3. Run the game:
```
python main.py
```

## 🎮 Controls
* ⬆️ Up Arrow → Move Up
* ⬇️ Down Arrow → Move Down
* ⬅️ Left Arrow → Move Left
* ➡️ Right Arrow → Move Righ
